# 🚀 System Design Journey

This repository documents my journey of learning and practicing **System Design and Distributed Systems**.

The goal of this repository is to build a strong understanding of designing **scalable, reliable, and production-grade systems**, similar to those used in large-scale tech companies.

---

# 🎯 Goals

* Learn **System Design fundamentals**
* Practice designing **real-world large-scale systems**
* Understand **scalability, reliability, and performance**
* Prepare for **SDE interviews in product-based companies**

# 📚 System Design Problems

This section contains system design case studies.

| Problem             | Description                         |
| ------------------- | ----------------------------------- |
| URL Shortener       | Design a service like TinyURL       |
| Rate Limiter        | Protect APIs from excessive traffic |
| Chat System         | Design WhatsApp / Messenger         |
| Notification System | Push notification service           |
| Distributed Cache   | Design Redis-like cache             |
| File Storage        | Design Dropbox / Google Drive       |
| Video Streaming     | Design YouTube / Netflix            |
| Ride Sharing        | Design Uber                         |
| Social Media Feed   | Design Instagram Feed               |

Each design includes:

* Requirements
* High Level Design
* Low Level Components
* Scaling Strategies
* Bottlenecks
* Improvements
